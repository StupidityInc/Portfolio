# Portfolio — urib.me

My personal website and portfolio. Built with vanilla HTML, CSS, and JavaScript.

🌐 **Live at:** [urib.me](https://urib.me)

## Tech Stack
- **HTML5** — Semantic structure
- **CSS3** — Custom styles with Grid, Flexbox, and animations
- **JavaScript** — Interactive elements
- **Font Awesome** — Icons
- **Google Fonts** — Syne + Outfit

## Features
- Responsive design (mobile-friendly)
- Animated color orbs and grid background
- Live time display in navigation
- Sections: Story, Focus, Contact

## Local Dev
Just open `index.html` in a browser — no build tools needed.
